  • 초대형 판매전문회사

    전국 450여개 지점의 영업망과 약 2만명 FP의
    판매조직을 보유한 국내 최대 판매전문회사입니다.

  • 고객 중심의 서비스

    500만 고객님께 더 나은 금융 서비스와 종합 솔루션을
    제공하고, 변함없는 신뢰를 받고 있는 금융파트너 입니다.

  • 도전과 혁신의 문화

    1946년 우리나라 최초로 생명보험을 시작한
    한화생명의 물적분할로 출범하여, 현장 중심 사고와
    유연한 조직 문화로 끊임없이 도전하고 혁신하는 기업입니다.
