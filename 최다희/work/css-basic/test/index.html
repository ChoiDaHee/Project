권민호:회색 숨

권민호 작가는 이번 프로젝트를 위해 옛 연초제조창이었던 국립현대미술관 청주 미술품수장센터 건축 도면 안에 연초제조창의 역사를 펼쳐 놓은 <회색 숨>을 제작했습니다. 권민호 작가의 작품 일부분을 튼튼한 타포린 원단에 특수 인쇄기법을 사용하여 생생하게 구현하였습니다. 카드의 QR코드를 인식하여 MMCA x Around 애플리케이션을 통해 국립현대미술관 청주의 사진을 비추면 작품을 3D 영상으로 감상할 수 있습니다.
